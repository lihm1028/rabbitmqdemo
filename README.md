# springboot 2.7.0 集成rabbitmq使用消息中间件MQ 

# springboot 使用rabbitmq
springboot可以通过amqp封装来直接操作Rabbitmq

```
        <!--rabbitmq-->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-amqp</artifactId>
        </dependency>
```
